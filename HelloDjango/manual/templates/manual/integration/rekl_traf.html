Traflead KZ, Traflead RU, Traflead UZ

Данные рекламодатели используют СРМ Leadgroup - аналог Leadvertex только с более старой версией API. (Если интеграция с этими СРМ, тогда каждое гео заводим отдельной кампанией).
Менеджер ОП скидывает в таск проект, в который нужно настраиваться.

{%img%} https://i.imgur.com/oAwM0EB.png {%end%}

Пример ссылки на проект (Доступ к проекту в документации)

{%code%} https://art438-sibericafree-kzpro.leadgroup.su/webmaster {%end%}

Идем во вкладку API И вносим апи ключ - его генерируем самостоятельно

{%img%} https://i.imgur.com/LkwzymB.png {%end%} {%note s%} Генерация ключа Переходим по ссылке, убираем галочку символы {%img%} https://i.imgur.com/32O0TFb.png {%end%} {%notice w%} По длине символов в генерируемом ключе лучше уточнить {%end%} {%end%}

Далее смотрим, чтобы разделы Получение списка заказов и Импорт заказов были открыты (присутстовали на странице).

В модуле отправки в адрес отправки вносим URL из раздела Импорт заказав

{%img%} https://i.imgur.com/MOwMwe5.png {%end%} {%img%} https://i.imgur.com/shwEUIl.png {%end%}

В поле additional1 вносим значение: webmaster_int - это означает, что в поле additional1 мы будем передавать реклу ID нашего вебмастера в числовом формате - числово-буквенный рекл не может принимать

В поле ID товара вносим ID товара из таска)

В модуле забора статусов вносим Адрес отправки из раздела Получение списка заказов . Далее, нужно будет подшаманить ссылку

{%img%} https://i.imgur.com/wQgl4nK.png {%end%} {%img%} https://i.imgur.com/6ncpuMI.png {%end%} {%notice w%} Важно ссылку берем до первого амперсанда {%end%}

Далее, нужно будет подшаманить ссылку:
